# 常见问题解答（FAQ）
本文汇总了使用 LLM-Toolkit 网站过程中的高频疑问，若未找到对应答案，可通过反馈渠道联系我们。

## 一、网站基础说明
### 1. 网站主要提供什么服务？
LLM-Toolkit 是一款 **AI 工具分类导航与对比平台**，目前已覆盖 43 个 AI 细分场景、700+ 款主流 AI 工具，核心提供工具分类查询、核心能力介绍、官网直达、工具对比等服务，助力用户快速找到适配自身需求的 AI 工具，无需逐个搜索筛选。

### 2. 网站收录的 AI 工具有什么特点？
收录工具均经过筛选，优先选择 **官网可正常访问、功能成熟、口碑良好** 的产品，涵盖通用大模型、AI 绘图、提示词工具、数据处理、编程辅助等全场景，同时区分开源/免费/付费工具，标注核心特色，方便用户按需选择。

### 3. 网站的工具会定期更新吗？
会定期更新，主要更新内容包括：新增热门 AI 工具、更新工具核心功能/官网链接、下架失效工具、优化工具分类与对比信息，确保工具信息的时效性和准确性。

### 4. 访问网站需要注册或付费吗？
不需要。网站完全免费开放，无需注册账号、无需付费订阅，所有工具信息、分类导航、对比内容均可直接查看，点击工具官网链接可直接跳转至对应平台。

## 二、工具使用相关
### 1. 如何快速找到我需要的 AI 工具？
有两种便捷方式：
- 1.导航栏：点击顶部导航栏，根据需求选择对应分类（如“AI 绘图工具”“AI 编程工具”），进入分类页面查看所有相关工具；
- 2.搜索框：使用网站顶部搜索框，输入工具名称、核心功能（如“ChatGPT”“提示词优化”），可快速定位目标工具。

### 2. 工具链接打不开/失效了怎么办？
若工具官网链接无法打开或提示失效，可点击页面底部的「反馈该分类工具的信息错误/链接失效」，跳转至 GitHub Issues 提交反馈，注明“工具名称+链接失效”，我们会在 1-3 个工作日内核实并更新。

### 3. 网站会对收录的 AI 工具进行测评吗？
目前网站主要聚焦「工具分类导航与核心信息展示」，暂不提供详细测评，但会客观标注每款工具的核心优势、适用场景，后续会逐步新增工具对比专区，提供更细致的功能、性价比对比。

### 4. 可以申请新增 AI 工具收录吗？
可以。进入导航栏「贡献与反馈」→「新增工具指南」，按照提示填写工具名称、核心能力、官网地址、适用场景等信息，提交后我们会在 1-3 个工作日内审核，审核通过后会收录至对应分类。

## 三、技术与部署相关
### 1. 网站部署在 GitHub Pages，加载缓慢怎么办？
加载缓慢主要受网络环境影响，可尝试以下解决方法：
1.  刷新页面，或更换浏览器（推荐 Chrome、Edge）；
2.  检查网络连接，切换稳定网络；
3.  若多次加载失败，可等待 10-20 分钟后再尝试（GitHub Pages 偶尔会有临时波动）。

## 四、反馈与贡献相关
### 1. 发现网站bug、内容错误，如何反馈？
可通过两种方式反馈：
1.  GitHub Issues：点击网站任意页面底部的反馈链接，跳转至 Issues 页面，详细描述 bug/错误内容（如“页面排版错乱”“工具核心能力描述错误”），并附上截图（可选）；
2.  仓库留言：在 GitHub 仓库主页的「Discussions」板块留言，我们会及时查看并回复。

### 2. 想参与网站优化，如何贡献？
欢迎通过以下方式参与贡献：
1.  提交工具信息更新、bug 修复：Fork 仓库，修改对应内容后提交 Pull Request，我们审核通过后会合并至主分支；
2.  提供优化建议：在 Issues 或 Discussions 中提交网站界面、功能、内容等方面的优化建议；
3.  新增工具收录：按照「新增工具指南」提交符合要求的 AI 工具。

### 3. 反馈后多久会得到回复/处理？
- 工具链接失效、内容错误反馈：1-3 个工作日内核实处理；
- 网站 bug 反馈：2-5 个工作日内排查修复；
- 新增工具、优化建议：3-5 个工作日内审核/回复。

## 五、其他常见问题
### 1. 网站支持移动端访问吗？
支持。网站已做响应式适配，可通过手机、平板等移动设备访问，导航栏、工具卡片会自动适配设备尺寸，确保移动端使用体验。

### 2. 网站收录的工具是否支持商用？
网站仅提供工具导航与信息展示，**工具的商用权限需参考对应工具官网的说明**，部分工具为免费商用、部分需付费授权、部分禁止商用，建议使用前仔细查看工具官网的版权/商用说明。

### 3. 如何查看网站的更新日志？
进入导航栏「审核日志」→「audit_log.md」，可查看网站的所有更新记录，包括工具新增、内容优化、bug 修复等，每条记录标注更新时间与更新内容。

### 4. 网站使用的主题、插件可以修改吗？（管理员专用）
可以。网站基于 MkDocs Material 主题搭建，可在 mkdocs.yml 配置文件中修改主题样式、新增/移除插件、调整导航栏结构，修改后提交 GitHub 并重新部署即可生效。
